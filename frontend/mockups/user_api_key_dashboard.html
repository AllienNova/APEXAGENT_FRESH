System Fallback Information

About System Fallback

When you don't provide your own API keys, Aideon AI Lite will use system keys to ensure uninterrupted service. This allows you to use all features even if you don't have keys for every service.

System keys are available for most popular services
Your own keys take priority when configured
Usage with system keys counts against your subscription credits

Benefits of Using Your Own Keys

  • Reduced subscription costs (operations with your keys don't use credits)
  • Direct control over API usage and billing
  • Access to your custom fine-tuned models
  • Higher rate limits based on your provider accounts
