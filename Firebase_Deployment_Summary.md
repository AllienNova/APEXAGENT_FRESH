## 🚀 Quick Deployment Guide

### 1. **Setup (5 minutes)**
```bash
# Extract the deployment package
tar -xzf Aideon_Firebase_Complete_Deployment.tar.gz
cd aideon_firebase

# Install dependencies
npm install
cd functions && npm install && cd ..
```

### 2. **Firebase Project Setup**
```bash
# Login to Firebase
firebase login

# Create/select project
firebase use --add

# Deploy everything
npm run deploy
```

### 3. **Go Live**
Your Aideon AI system will be live at:
`https://YOUR-PROJECT-ID.web.app`
